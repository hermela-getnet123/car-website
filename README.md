# Car Website 🚗

A responsive car showcase website developed using HTML, CSS, and JavaScript.

The project displays different car models with a clean and modern interface designed for practicing frontend web development skills.

---

## 🚀 Features

- Responsive web design
- Modern UI layout
- Car showcase sections
- Navigation bar
- Interactive styling and hover effects

- JavaScript
